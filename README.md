# RoomSensor
Temperature and humidity sensor for bedroom.

# Raspberry Pi Pico Pinouts

## Pico Power
3.3V breadboard --> VSYS Pin 39

GND breadboard --> Pin 38

## DHT11 Temp/Humidity Sensor
VCC --> 3V3 breadboard

GND --> GND breadboard

Signal --> GPIO28 (Pin 34)

## LCD Screen
LCD VCC --> VBUS (Pin 40)

LCD GND --> GND of breadboard

SDA --> Pin 6

SCL --> Pin 7
